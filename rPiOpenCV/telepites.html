Raspberry Pi telepítése

A https://www.raspberrypi.org/software/ oldalról letölthető az a szoftver, ami lehetővé teszi az operációs rendszer kiválasztását és SD kártyára írását. Én az alapértelmezett Raspberry Pi 32 bites OS-t választottam, ennek mérete kb.: 1.2 Gb

A következő lépéshez - ami az OpenCV telepítése - hozzá kellene férni a Raspberry Pi-n futó operációs rendszerhez. Ennek egyik módja az is lehetne, hogy csatlakoztatok az eszközhöz monitort, billentyűzetet és esetleg egeret. Azonban ezeket a Raspberry-ket a leggyakrabban távoli hozzáféréssel kezelik, én is ezzel próbálkoznék. Némi keresés után az alábbi cikket találtam: https://desertbot.io/blog/headless-raspberry-pi-4-ssh-wifi-setup.

Windows-ról a csatlakozáshoz használható a Putty nevű program, azonban én korábban már telepítettem a WSL-t (Info a WSL-ről ) a gépemre, ezért ezt használom. A következő akadály: mi a Raspberry IP címe? Több módszer létezik a legyeszerűbb a Router-en megnézni a kiosztott IP címeket, valamint még mindig opció rádugni a monitor, billenytyűzet, egér hármast.
A csatlakozáshoz érdemes tudni, hogy az alapértelmezett felhasználónév pi, az alapértelmezett jelszó pedig raspberry.

Érdemes egy frissítéssel kezdeni: 

sudo apt-get update && sudo apt-get upgrade -y && sudo rpi-update
Érdemes felkészülni arra, hogy az upgrade akár 15-20 percig is tarthat. A folyamat végén újra kell indítani az eszközt: 
sudo reboot

Eztuán ismét némi keresgélés, hogyan kellene az OpenCV-t telepíteni. Több cikket elolvasva, az alábbi lépések végrehajtása után sikerült az OpenCV-t telepíteni: 

sudo apt install cmake pkg-config build-essential git -y
sudo apt install libwebp-dev libjasper-dev libopenexr-dev libv4l-dev libavformat-dev -y
sudo apt install libxvidcore-dev libx264-dev libdc1394-22-dev libgstreamer-plugins-base1.0-dev libgstreamer1.0-dev libavcodec-dev -y
sudo apt install libtiff-dev libpng-dev libjpeg-dev libswscale-dev -y
sudo apt install libhdf5-dev libhdf5-103 libatlas-base-dev -y
sudo apt-get install libssl-dev -y
sudo apt install libgtk-3-dev libqtwebkit4 libqt4-test libqtgui4 python3-pyqt5 -y
sudo apt install python3-dev python3-pip python3-testresources python3-numpy -y
Az utolsó két utasítás már a python telepítőjével történik 
sudo pip3 install -U numpy
sudo pip3 install opencv-python
